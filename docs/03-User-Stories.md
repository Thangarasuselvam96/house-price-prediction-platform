# User Stories

## Epic 1: User Management

### US-001 - User Registration

**As a new user,**
I want to create an account
**so that** I can access the platform.

---

### US-002 - User Login

**As a registered user,**
I want to log in securely
**so that** I can access my dashboard.

---

### US-003 - Update Profile

**As a logged-in user,**
I want to update my profile information
**so that** my details remain accurate.

---

## Epic 2: Property Management

### US-004 - Add Property

**As a seller,**
I want to create a property listing
**so that** buyers can view my property.

---

### US-005 - Edit Property

**As a seller,**
I want to edit my property details
**so that** the information stays up to date.

---

### US-006 - Delete Property

**As a seller,**
I want to remove my property listing
**so that** unavailable properties are no longer displayed.

---

## Epic 3: Property Search

### US-007 - Search Properties

**As a buyer,**
I want to search properties by location and filters
**so that** I can find properties matching my needs.

---

### US-008 - View Property Details

**As a buyer,**
I want to view complete property information
**so that** I can make an informed decision.

---

### US-009 - Save Favorite Property

**As a buyer,**
I want to save interesting properties
**so that** I can view them later.

---

### US-010 - Contact Seller

**As a buyer,**
I want to send an enquiry to a seller
**so that** I can learn more about a property.

---

## Epic 4: House Price Prediction

### US-011 - Predict Property Price

**As a buyer or seller,**
I want to enter property details and receive an estimated market price
**so that** I can make better pricing decisions.

---

## Epic 5: Administration

### US-012 - Approve Listings

**As an administrator,**
I want to review and approve property listings
**so that** only valid listings are published.

---

### US-013 - Manage Users

**As an administrator,**
I want to manage user accounts
**so that** the platform remains secure and trustworthy.

---

## Epic 6: Dashboards

### US-014 - Buyer Dashboard

**As a buyer,**
I want to view my saved properties and enquiries
**so that** I can manage my activity.

---

### US-015 - Seller Dashboard

**As a seller,**
I want to view my listed properties and enquiries
**so that** I can monitor my listings.

---

### US-016 - Admin Dashboard

**As an administrator,**
I want to view platform statistics and pending approvals
**so that** I can efficiently manage the platform.
